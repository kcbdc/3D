# 모듈별 구현 결과

1. 에셋 모듈
   - 첨부된 낮·밤 배경과 훈민·다임·순식 PNG 적용
   - 첨부 캐릭터 선택 시안 적용

2. 월드·경로 모듈
   - `src/config/world-data.js` 하나에서 유도선, 충돌, 자동 이동을 모두 생성
   - 보이는 유도선과 실제 이동 영역이 동일함
   - 건물과 물 통과 차단, 다리 구간만 물 횡단

3. 게임 시스템
   - 업무 → 골드 → 씨앗 구매 → 심기 → 성장 → 수확
   - 캐릭터별 이동·보상 특성

4. UI
   - 퀘스트 패널만 상시 표시
   - 기타 상시 팝업 제거
   - 상점과 농장은 상호작용 시에만 모달 표시

5. 시간 전환
   - 현지 브라우저 시간 06:00~17:59: 낮 배경
   - 18:00~05:59: 밤 배경

6. 모바일 가로 고정
   - PWA manifest orientation=landscape
   - 게임 시작 시 Screen Orientation API 시도
   - 세로 기기에서는 게임 쉘을 CSS로 90도 회전
