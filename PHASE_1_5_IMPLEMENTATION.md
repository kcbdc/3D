# Phase 1~5 통합 구현본

## Phase 1 월드 리빌드
- 배경 속 고정 캐릭터 제거
- 도로/인도 이동 네트워크
- 건물 충돌
- 강 충돌
- 다리 구간만 횡단 가능
- 실제 배경 위치와 상호작용 좌표 동기화
- `?debug=1` 이동 마스크 표시

## Phase 2 캐릭터
- 변경 시안 기반 훈민/다임/순식 이미지 교체
- 방향 전환, 보행 바운스, 그림자
- 캐릭터별 특성

## Phase 3 도시 시스템
- NPC 이동
- 자동차 이동
- 네온/비 효과
- 도시 핫스팟

## Phase 4 게임성
- 업무, 상점, 농장
- 인벤토리, 장비
- 퀘스트, 업적, 랭킹 점수
- 저장/불러오기 확장

## Phase 5 출시 준비
- Cloudflare Pages Functions
- D1 스키마
- 로그인 연결 UI
- 서버 시간 검증
- PWA 서비스워커
- 모바일 조이스틱 최적화

Cloudflare Pages: Framework None / Build command 공란 / Output `/`
