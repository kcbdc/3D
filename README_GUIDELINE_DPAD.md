# 기준 시안 반영 최종 수정본

## 개선사항

- 숨겨진 이동 경계 대신 모든 도로에 청색 애니메이션 유도선 표시
- 유도선 위 방향 화살표 표시
- 다리 구간은 더 밝은 청록색으로 강조
- 캐릭터 주변에 남은 카드 배경과 반투명 사각형 제거
- 기준 시안과 같은 상·하·좌·우 방향키 패드 적용
- PC 방향키/WASD 및 AUTO 길찾기 유지
- 건물·물 통과 차단, 다리에서만 강 횡단
