# 네온시티 실제 반영 재검증본

이 버전은 이전처럼 UI만 바꾼 버전이 아닙니다.

## 실제 변경된 부분

- 기존 `diamond`, `prism`, 절차형 아이소메트릭 건물 렌더링을 게임 월드에서 제거
- 승인된 네온 도시 시안에서 실제 플레이 배경 이미지 분리
- 승인된 영웅 선택 시안에서 실제 캐릭터 선택 배경 이미지 분리
- 도시 배경 위에서 고해상도 캐릭터가 이동
- 본사·제지본부·기술연구원·화폐본부·씨앗상점·주말농장 핫스팟 배치
- 핫스팟 접근 시 업무·상점·농장 기능 실행
- 미션·가방·지도·상점·랭킹·도감 UI 유지
- PC 키보드와 모바일 조이스틱 이동 지원
